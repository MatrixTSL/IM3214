CP6211 Industrial Maintenance - Sense and Control Worksheets

11 comprehensive training worksheets covering PLC programming, control systems, sensors, and industrial automation concepts.

Worksheet 1: Basic Outputs

Modern industrial control systems including low-current transistor outputs and high-current relay outputs.

Worksheet 2: Sequenced Outputs

Switching between system states and creating state machines for traffic light control systems.

Worksheet 3: Pulse Width Modulation

Motor speed control using PWM techniques and FET current amplifiers for industrial applications.

Worksheet 4: Basic Inputs

Understanding digital and analogue sensors, two-state outputs, and industrial proximity sensors.

Worksheet 5: Pedestrian Crossing

Fully-functional road-crossing system combining timers, state machines, and traffic control logic.

Worksheet 6: Potentiometers

Variable voltage control for setting light levels, conveyor speed, and temperature controls.

Worksheet 7: Using Sensors

Digital and analogue sensors including thermistors for temperature monitoring and alarm systems.

Worksheet 8: Detecting Faults 1

Landing gear fault detection using current monitoring and broken bulb detection systems.

Worksheet 9: Detecting Faults 2

Advanced fault detection for analogue sensors including short-circuit and signal validation.

Worksheet 10: Open-Loop Control

Understanding open-loop control systems with no feedback for motor speed control applications.

Worksheet 11: Closed-Loop Control

Closed-loop control systems with feedback for light level control and environmental monitoring.
