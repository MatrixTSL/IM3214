Worksheet 10 - Open-Loop Control

1. Introduction

There are two principal types of control system: open loop and closed loop. Closed loop systems have some kind of feedback which allows for system regulation. Open loop systems have no feedback and engineers rely on an understanding of the components in the system to provide the required functionality.

3. So What?

Open-Loop Control System

This is an example of an open-loop control system. Open-loop systems control outputs without feedback. They are simple but won't detect faults like a stuck valve or failed actuator.

PWM Motor Control

Pulse Width Modulation (PWM) varies motor speed by changing the ratio of on-time to off-time. The system assumes the motor speed matches the PWM setting, but has no way to verify actual speed.

Industrial Applications

Knowing where these are used helps identify issues during inspections and repairs. Open-loop systems are common in conveyor belts, fans, and simple pump applications where exact speed isn't critical.

Key Takeaways

  • PLCs use digital outputs to control devices like lights, motors, and valves.
  • Outputs can be triggered by buttons, switches, sensors, or analogue signals.
  • Latching and selector switches show different output behaviours.
  • Sensors like proximity and analogue inputs can drive outputs.
  • Sequences (“dance mode”) mirror real process timing patterns.

Questions & Answers

Question 1: What is the main characteristic of an open-loop control system?

Correct Answer: C) It controls outputs without feedback from the process.

Question 2: How does PWM (Pulse Width Modulation) control motor speed?

Correct Answer: B) By varying the ratio of on-time to off-time in pulses.

Question 3: What is a limitation of open-loop control systems?

Correct Answer: B) They cannot detect faults like a stuck valve or failed actuator.

Question 4: What does the selector switch (I4/I5) control in this worksheet?

Correct Answer: A) It changes the behaviour of LED Q3. Left keeps it on, right makes it flash.

Question 5: Why is it useful to recognise output patterns like the LED sequence?

Correct Answer: A) Because real systems often use sequences and recognising them helps with fault finding and confirming correct operation.
