Worksheet 10 - Open-Loop Control

1. Introduction

An open-loop control system controls outputs without feedback from the process. It assumes the output device works as commanded but cannot verify actual performance. This is simple to program but cannot detect faults like a stuck valve or failed actuator.

3. So What?

Open-Loop Control System

This is an example of an open-loop control system. Open-loop systems control outputs without feedback. They are simple but won't detect faults like a stuck valve or failed actuator.

PWM Motor Control

Pulse Width Modulation (PWM) varies motor speed by changing the ratio of on-time to off-time. The system assumes the motor speed matches the PWM setting, but has no way to verify actual speed.

Industrial Applications

Knowing where these are used helps identify issues during inspections and repairs. Open-loop systems are common in conveyor belts, fans, and simple pump applications where exact speed isn't critical.

Key Takeaways

  • Open-loop systems control outputs without feedback from the process.
  • PWM (Pulse Width Modulation) varies motor speed by changing pulse ratios.
  • Open-loop systems assume outputs work as commanded - no verification.
  • Simple to program but cannot detect faults automatically.
  • Common in conveyor belts, fans, and simple pumps where exact control isn't critical.
