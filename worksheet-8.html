3. So What?

Safety-Critical Systems

In safety-critical systems, extra circuitry is often used to confirm that devices are operating as expected. This prevents situations where a command is sent but the device fails to respond.

Current Monitoring Method

One common method is placing a small resistor in line with an actuator. By measuring the voltage across the resistor, the system can detect if current is flowing.

Operational Verification

This helps verify that the actuator is not just being commanded, but is actually drawing power and likely operating correctly. Essential for aircraft, nuclear, and process safety systems.

Key Takeaways

  • Fault detection systems monitor actual device operation, not just commands.
  • Current monitoring through resistors verifies electrical load operation.
  • Safety-critical systems require verification that devices respond as expected.
  • Broken bulb/device detection prevents unnoticed system failures.
  • Analogue monitoring provides real-time feedback on device health.
